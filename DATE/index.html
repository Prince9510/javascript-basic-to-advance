ðŸš€ Creating Date Objects (Constructors)

The Date object is used to work with dates and times. A new instance is created using the new Date() constructor.

Syntax Description
new Date() Creates a date object with the current date and time.
new Date(milliseconds) Creates a date object from milliseconds since the UNIX Epoch (Jan 1, 1970).
new Date(dateString) Creates a date object from a date string (e.g., "2025-08-12").
new Date(year, monthIndex, day, hours, minutes, seconds, ms) Creates a date object with the specified date and time components. Note: monthIndex is 0-11.

ðŸ“¥ Getter Methods (Get Date Components)

These methods retrieve information from a date object. They return values based on the user's local time zone.

Method Description Return Value Example
getFullYear() Gets the 4-digit year. 2025
getMonth() Gets the month (0-11; January is 0). 7 (for August)
getDate() Gets the day of the month (1-31). 12
getDay() Gets the day of the week (0-6; Sunday is 0). 2 (for Tuesday)
getHours() Gets the hour (0-23). 10
getMinutes() Gets the minute (0-59). 42
getSeconds() Gets the second (0-59). 55
getMilliseconds() Gets the millisecond (0-999). 123
getTime() Gets the number of milliseconds since the UNIX Epoch. 1754985175000

ðŸ“¤ Setter Methods (Set Date Components)

These methods are used to set parts of a date in an existing date object.

Method Description
setFullYear(year, [month], [day]) Sets the 4-digit year.
setMonth(monthIndex, [day]) Sets the month (0-11).
setDate(day) Sets the day of the month (1-31).
setHours(hour, [min], [sec], [ms]) Sets the hour (0-23).
setMinutes(min, [sec], [ms]) Sets the minute (0-59).
setSeconds(sec, [ms]) Sets the second (0-59).
setMilliseconds(ms) Sets the millisecond (0-999).
setTime(milliseconds) Sets the date using milliseconds since the UNIX Epoch.

ðŸ”„ Conversion & String Methods

These methods convert a date object into a more readable string format.

Method Description
toString() Returns the full date as a string.
toDateString() Returns the date portion as a human-readable string.
toTimeString() Returns the time portion as a human-readable string.
toLocaleString() Returns the full date as a string, formatted for the local region (e.g., 12/8/2025, 10:42:55 am).
toLocaleDateString() Returns the date portion as a string, formatted for the local region (e.g., 12/8/2025).
toLocaleTimeString() Returns the time portion as a string, formatted for the local region (e.g., 10:42:55 am).
toISOString() Returns the date in ISO 8601 format (YYYY-MM-DDTHH:mm:ss.sssZ).
toUTCString() Returns the date as a string, formatted according to the UTC time standard.
toJSON() Returns a string in ISO format, suitable for JSON serialization.

âš¡ Static Methods

These methods are called directly on the Date class itself, not on an instance.

Method Description
Date.now() Returns the number of milliseconds since the UNIX Epoch (same as new Date().getTime()).
Date.parse(dateString) Parses a date string and returns the milliseconds since the UNIX Epoch.
Date.UTC(year, monthIndex, ...) Accepts date components and returns the milliseconds since the UNIX Epoch, assuming UTC.
